# PROJECT CONTEXT

## Project

Modern ERP Accounting System

---

## Background

Sistem ini dibangun untuk perusahaan distributor bahan bangunan.

Project ini bukan clone ERPNext.

Project ini juga bukan clone sistem lama.

ERP lama hanya dijadikan referensi user experience dan workflow yang dapat diamati.

ERPNext dijadikan referensi business rule, accounting, inventory, purchasing, selling, permission, dan konsep ERP.

---

Frontend

## Goal

Membangun ERP yang:

- lebih sederhana
- modern
- mudah digunakan
- mudah dikembangkan
- tetap mengikuti prinsip ERP
